Detailed Description

Class responsible for calculating forces applied to graph nodes. It also contains method for rearranging nodes on corresponding Canvas CalculateForces method ivokes only one "tick" of arrangement alghoritm, so you should use it in loop or timer multiple times (more than 100 recommneded).
